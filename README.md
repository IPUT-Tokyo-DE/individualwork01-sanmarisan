# よけろ!!

## ゲームの内容
制限時間内で四角形をよけながらなるべく多くの三角形をとる
三角形を取るとスピードアップ、逆に四角形を取ってしまうとスピードダウン
スピードダウンしすぎてしまうと動けなくなる
